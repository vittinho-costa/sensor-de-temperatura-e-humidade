# sensor-de-temperatura-e-humidade
Sistema de Monitoramento de Solo (UFPA) utilizando PIC16F877A. Desenvolvido para medição de temperatura e umidade com interface LCD e teclado. Permite configurar intervalos e número de amostras, armazenando dados na EEPROM. Possui alarme sonoro de conclusão. Projeto didático focado em sistemas embarcados e automação agrícola de baixo custo.
